使用手册

初次使用

  1. 将设备连接到 5V 电源适配器
  2. 设备通电后自动启动，显示屏显示启动信息
  3. 按 KEY1 按键启动系统
  4. 等待设备扫描 WiFi 网络（约 5 秒）
  5. 在显示屏上选择目标 WiFi 并输入密码连接

操作指南

1. 系统开关 (KEY1)

操作：按一下 KEY1
功能：控制整个系统的开启与关闭

2. 模式切换 (KEY2)

操作：长按 KEY2 超过 2 秒
功能：在手动模式和自动模式之间切换

3. 手动模式控制 (KEY2)

在手动模式下，短按 KEY2 可循环切换四种状态：

  • 状态 00：灯光 OFF，风扇 OFF
  • 状态 01：灯光 OFF，风扇 ON
  • 状态 10：灯光 ON，风扇 OFF
  • 状态 11：灯光 ON，风扇 ON

4. 自动模式说明

在自动模式下，设备根据传感器数据自动控制：

  • 灯光控制：光照强度 < 设定阈值时自动点亮
  • 风扇控制：温度 > 设定阈值时自动开启

MQTT 远程控制

设备成功连接 WiFi 后，会自动尝试连接 MQTT 服务器。连接成功后，可通过以下 MQTT 主题进行远程控制：

主题：esp32c3/control/light

有效 payload：ON / OFF


主题：esp32c3/control/fan

有效 payload：ON / OFF


主题：esp32c3/control/mode

有效 payload：AUTO / MANUAL


主题：esp32c3/status

发布内容：{"light":"ON","fan":"OFF","temperature":25.5,"light_intensity":281,"mode":"AUTO"}

⚠️ 注意：远程控制仅在 MQTT 连接成功后生效，显示屏会显示 [MQTT] 标识

常见问题

Q1：设备无法连接 WiFi？

A：请确保 WiFi 密码正确，设备支持 2.4GHz WiFi 网络，不支持 5GHz。

Q2：MQTT 连接失败？

A：请检查网络连接是否正常，确认 MQTT 服务器地址和端口配置正确。

Q3：传感器数据显示异常？

A：请检查传感器接线是否正确，重新启动设备尝试。

Q4：按键无响应？

A：请确认系统已启动（按 KEY1），检查按键接线是否松动。
